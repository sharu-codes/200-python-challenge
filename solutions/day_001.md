# Day 001 - Even or Odd Number Checker

**Topic:** Numbers, Conditionals  
**Difficulty:** Easy  
**Date:** 04-06-2026

## Problem

Given a number, check whether it is **Even** or **Odd**.

- **Even numbers:** divisible by 2 (remainder = 0)  
  Example: 2, 4, 6, 8, 10

- **Odd numbers:** NOT divisible by 2 (remainder = 1)  
  Example: 1, 3, 5, 7, 9

## Concept

**Modulus operator `%`** — gives the remainder after division.

```
10 % 2 = 0  → Even
7 % 2 = 1   → Odd
```

## Solution Steps

1. Take a number as input
2. Divide it by 2 and check the remainder
3. If remainder is 0 → Even, else → Odd
4. Print the result
