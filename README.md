## How to Use This Repo

1. **View Solutions:** Click any `pXXX.py` file to see the code with detailed comments
2. **Understand Concepts:** Each solution includes:
   - Problem statement
   - Step-by-step explanation
   - Code with comments
   - Example outputs
3. **Run Locally:** 
   ```bash
   cd solutions
   python p001.py
   ```
